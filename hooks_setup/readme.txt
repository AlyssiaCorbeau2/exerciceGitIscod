In order to do the exercise, I have to make this message explaining, to new
developpers on my project, how to setup the hooks of this directory on their
image of the project.

1 : copy the no-extension files from this directory
2 : paste them into .git/hooks, .git can be found at the root of the project

Voilà, you are now as clueless than me on the utility of this hook ^^.
You also will be clueless on the warning it shows when using git commit :)
